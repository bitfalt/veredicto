# ElevenLabs x Firecrawl Starter

Prompt-first starter for building an ElevenHacks product with `ElevenAgents` and `Firecrawl Search`.

## Quick start

1. Copy `.env.example` to `.env.local`
2. Fill in your API keys
3. Install dependencies with `npm install`
4. Run the app with `npm run dev`

## Included

- `Next.js` + `TypeScript` + `Tailwind CSS`
- Signed URL API route for ElevenLabs
- Firecrawl search API route
- Starter landing page
- Prompt-first folders for docs, prompts, remotion, scripts, and SDK wrappers
